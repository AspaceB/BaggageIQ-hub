Privacy Policy

BaggageIQ — Smart Flight Pricing with Baggage

Overview

BaggageIQ is a browser extension that helps travelers see the real cost of flights by adding estimated baggage fees to displayed prices. We are committed to protecting your privacy and being transparent about what data we handle.

Data We Collect

We do not collect, transmit, or store any personal data. Specifically:

  • No browsing history is collected or shared
  • No flight search details leave your browser
  • No user accounts or sign-ins are required
  • No analytics, tracking pixels, or third-party scripts are used
  • No cookies are set by the extension

Local Storage

BaggageIQ uses Chrome's storage.local API solely to save your preferences:

  • Extension enabled/disabled state
  • Number of cabin and checked bags selected
  • Scoring priority preference (cost, time, or balanced)

This data is stored locally on your device and is never transmitted to any server.

Network Requests

The extension makes a single network request to fetch the latest airline baggage database from our public GitHub Pages URL (aspaceb.github.io/BaggageIQ-hub). This request:

  • Contains no personal information
  • Is a simple GET request for a static JSON file
  • Falls back to the bundled database if the request fails

Host Permissions

The extension requires access to specific travel websites (Skyscanner, MakeMyTrip, Kayak, Google Flights, CheapOAir) to read flight prices displayed on those pages and overlay baggage cost information. It does not access any other websites or read any data beyond what is displayed on flight search results pages.

Feedback Forms

If you voluntarily submit feedback through our website, any information you provide (name, email, message) is sent via FormSubmit.co to our email. This data is used solely to respond to your feedback and is not shared with third parties.

Third-Party Services

The extension itself uses no third-party services. The landing page website uses:

  • Google Fonts (for the Inter typeface)
  • FormSubmit.co (for processing feedback form submissions)

Changes to This Policy

If we make material changes to this privacy policy, we will update this page and change the date below.

Contact

If you have questions about this privacy policy, please reach out at support@baggageiq.app.
