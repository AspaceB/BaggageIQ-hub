Install

Get BaggageIQ running in under a minute

  1. Download the extension from the Chrome Web Store (coming soon) or load it manually
  2. Open chrome://extensions/ in Chrome
  3. Enable Developer mode (top-right toggle)
  4. Click Load unpacked and select the extension folder
  5. Pin the BaggageIQ icon in your toolbar — done!
